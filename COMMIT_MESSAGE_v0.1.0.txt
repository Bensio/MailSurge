Release v0.1.0 - Enterprise Contact Library & Email Campaign Management

## Major Features

### Enterprise Contact Library
- Tabbed interface (Library / All / Campaigns) for better organization
- Real-time search and filtering across all contacts
- Bulk operations: select, delete, move to library, export CSV
- Automatic deduplication of library contacts
- Campaign association tracking

### Email Campaign Management
- Visual email editor with Unlayer
- Multiple Gmail account support
- Campaign sending with configurable delays
- Retry failed campaigns functionality
- Archive completed campaigns
- Real-time progress tracking

### Gmail Integration
- OAuth 2.0 authentication
- Multiple account support
- Automatic token refresh
- Direct email sending via Gmail API

### Contact Management
- Individual contact addition
- CSV bulk upload
- Library-based contact storage
- Add contacts from library to campaigns
- Status tracking (pending, sent, failed)

## Technical Improvements

- Centralized logging utility (development vs production)
